# PrepX User Journeys

## 1. First-Time Aspirant Journey (0 → Prelims → Mains → Interview)

```
Discover (Day 0)
│
├─ Sees PrepX ad on Instagram / YouTube
├─ Lands on / (Landing Page)
│  ├─ Hero: "Clear UPSC with your AI Mentor"
│  ├─ Social proof: "12,000+ serious aspirants trust PrepX"
│  └─ CTA: "Start Free — No Credit Card"
│
├─ Taps CTA → /onboarding
│
Onboarding (Day 0, 3 min)
│
├─ Screen 1: "What's your goal?" — IAS / IPS / IFS / IRAS
├─ Screen 2: "Your background" — Graduate / Working / Repeater
├─ Screen 3: "Your prep level" — Self-assessment sliders
│   ├─ Polity: 1-10
│   ├─ History: 1-10
│   ├─ Economy: 1-10
│   └─ Current Affairs: 1-10
├─ Screen 4: "Your schedule" — Hours per day
│
├─ AI generates Personalized Roadmap (animated, 5s)
│  └─ "Welcome, Aarav. Your 12-month plan is ready."
│
└─ Redirects to /home

─────────────────────────────────────────────

First Week (Days 1-7)
│
├─ Opens /home at 7 AM
│  ├─ Daily Progress Ring: 0/5 tasks
│  ├─ Today's Plan card: "Polity Fundamental Rights"
│  ├─ AI Coach suggestion: "Start with today's topic"
│  └─ Quick Action: "Start Polity Quiz"
│
├─ Taps /action → sees task list
├─ Taps task → /topic/[tag] (Topic Viewer)
│  ├─ Reads content (EN). Content is short, chunked.
│  ├─ Switches to HI — sees Devanagari text
│  └─ Taps "Take Quiz"
│
├─ /quiz/[id]
│  ├─ 5 MCQs, 60s timer per question
│  ├─ Gets 3/5 correct
│  ├─ Score animation slides in with gentle bounce
│  ├─ View explanations → /dashboard
│
├─ /dashboard
│  ├─ Sees accuracy ring: 60%
│  ├─ "Your weak area: Fundamental Rights"
│  └─ Motivation: "Next revision in 2 days"
│
└─ Day 7: Unlocks Study Squad invitation

─────────────────────────────────────────────

Month 1-3 (Building The Habit)
│
├─ Opens daily at 7 AM → /home
│  ├─ 7-day streak badge flares
│  ├─ AI Coach nudges: "You are 12% ahead of plan"
│  └─ Today's tasks auto-scheduled by Hermes
│
├─ Weekend: /race (UPSC Race)
│  └─ Competes with 50 aspirants → ranks 12th
│
├─ Uses /blueprint for first time
│  └─ Explores full syllabus tree
│     ├─ GS Paper 1 → History → Ancient → Indus Valley
│     └─ Clicks node — sees progress bar
│
├─ Day 14: Receives Day 14 Reveal
│  └─ Predicted rank: AIR 340 (if exam today)
│     └─ Motivation spark: "You are on track for top 500."
│
└─ Month 3: Accuracy trend shows 60% → 74%

─────────────────────────────────────────────

Month 6 (The Plateau)
│
├─ Emotional dip — anxiety sets in
├─ Nudge: "Aarav, your consistency is your edge."
├─ Taps weak area from /dashboard
├─ Reattempts past quizzes — streak builds back
├─ /interview → first mock interview
│  └─ AI panel asks 5 questions, scores 6.5/10
│     └─ "Improve: Use more examples"
│
└─ Joins Study Squad for mutual accountability

─────────────────────────────────────────────

Month 12 (Pre-Exam)
│
├─ Daily mocks: /quiz with UPSC-length paper
├─ /predictions shows high-confidence topics
├─ Last /reveal before exam: AIR 120 predicted
├─ Push notification: "You have prepared 400+ hours."
│
└─ Enters exam with confidence

─────────────────────────────────────────────

Post-Result (Success / Retry)
├─ If cleared: Alumni network access, optional coaching marketplace
├─ If not: /blueprint resets with gap analysis
│  └─ "Your accuracy in Polity was 82% but Geography only 61%"
│     └─ New plan auto-generated
```

**Emotional Arc**: Curious → Confident (Day 14) → Anxious (Month 6) → Resilient → Proud/Determined

**Pain Points**:
- Week 2: Not seeing visible progress → Counter: streaks + ring animations
- Month 3: Too much syllabus → Counter: Blueprint hides unscheduled topics
- Month 6: Burnout fear → Counter: nudges + Study Squads

**Delight Moments**:
- First roadmap generation animation
- First Day 14 Reveal cinematic reveal
- Beating previous rank in UPSC Race

## 2. Repeater Journey (Targeted Improvement)

```
Entry: /onboarding
├─ Selects "Repeater"
├─ Skips basic questions (fast-track)
├─ Uploads previous marksheet (optional)
├─ Inputs last attempt weak areas
│
├─ AI generates GAP ANALYSIS instead of full roadmap
│  └─ "You need +30 marks in GS Paper 3"
│
└─ Redirects to /home

─────────────────────────────────────────────

Daily Use (Efficiency-Focused)
│
├─ /home shows "Focus Areas" instead of generic plan
│  ├─ Today's focus: Environment + Science & Tech
│  └─ Quick Action: "Re-attempt failed questions"
│
├─ /dashboard
│  ├─ Compare with previous attempt graph
│  ├─ Time-to-answer trend: improving
│  └─ Heatmap: red zones from last year now orange
│
├─ /action
│  └─ Task list filtered to "Gap Items" only
│
├─ /interview
│  └─ Mock interview with "UPSC-style stress questions"
│     └─ Real-time scoring overlay
│
└─ Uses /predictions as final-week checklist
```

**Emotional Arc**: Frustrated (from previous failure) → Hopeful (gap analysis) → Focused → Confident

**Pain Points**:
- Repeating same content → Counter: Only weak areas shown by default
- Time guilt → Counter: "You only need 2 hours today" nudges

## 3. Working Professional Journey (Time-Constrained)

```
Entry: /onboarding
├─ Selects "Working Professional"
├─ Sets available hours: "1.5 hours weekdays, 4 hours weekends"
├─ Selects preferred study slots: 7-8:30 AM, 9-11 PM
│
├─ AI generates compressed plan
│  └─ "48-week plan for working professionals"
│     └─ Shorter daily tasks, intensive weekends
│
└─ Redirects to /home

─────────────────────────────────────────────

Daily Use (Mobile-First, Commute Mode)
│
├─ 7 AM push: "Today's micro-session: 15 min Polity revision"
├─ Opens /home on phone during commute
│  ├─ Tasks are bite-sized (10-15 min each)
│  └─ Audio mode available for theory content
│
├─ /action during lunch break
│  └─ Quick 10-question quiz (5 min)
│
├─ Evening: /method
│  └─ Views weekend plan — "4 hours Saturday: Full mock"
│
├─ Weekend /race
│  └─ Mobile-optimized speed quiz
│
└─ /dashboard weekly review (Sunday, laptop)
   └─ Sees cumulative progress bar
```

**Emotional Arc**: Overwhelmed → Relieved (bite-sized plan) → Consistent → Proud of discipline

**Pain Points**:
- Missing days due to work → Counter: "Catch-up" mode suggests compressed revision
- Too tired for quizzes → Counter: "Audio revision" mode

## 4. Daily Active User Journey (Ideal Retention Loop)

```
Morning (7:00 AM)
├─ Push notification: "Good morning, Aarav. 3 tasks today."
├─ Opens /home (2.3s load, skeleton → content)
│  ├─ Progress ring: 0% → fills as day progresses
│  ├─ Streak counter flares (if ≥3 days)
│  ├─ AI Coach: "Your Polity revision is overdue. 10 min?"
│  └─ Quick Actions: [Start Quiz] [Read Topic] [Daily Plan]
│
├─ Decision Point:
│  ├─ If morning: taps Daily Plan → /method
│  └─ If commute: taps Quiz → /quiz

─────────────────────────────────────────────

Mid-Morning (10:00 AM)
├─ Taps /topic/[id] from /action
│  ├─ EN mode by default
│  ├─ Reads chunked content (scroll), switches to HI
│  ├─ Mind map view toggle (desktop)
│  └─ Bookmarks for later
│
└─ Taps "Take Quiz"

─────────────────────────────────────────────

Mid-Day (1:00 PM)
├─ /quiz/[id]
│  ├─ Question card slides up
│  ├─ Selects option → instant visual feedback (correct: green pulse, incorrect: red shake + reveal correct)
│  ├─ 5 questions complete
│  ├─ Score animation: 4/5 = 80%
│  ├─ Confetti burst on >80%
│  └─ Swipes to explanations
│
└─ Back to /home → Progress ring now at 60%

─────────────────────────────────────────────

Evening (8:00 PM)
├─ Opens /dashboard
│  ├─ Accuracy graph updated
│  ├─ Weak area alert: "Geography: 55% — try revision?"
│  └─ Consistency heatmap: today = green
│
├─ Optional: /interview (10 min mock)
├─ Optional: /race (compete)
│
└─ Close: Streak counter + tomorrow preview
```

**Key Metrics**: 4 sessions/day, 18 min avg session, >75% daily task completion

## 5. Admin / Operator Journey

```
Entry: Direct /admin + Role Gate
│
├─ /admin Dashboard
│  ├─ Top cards: Total users, Active today, Revenue this month
│  ├─ Chart: User retention cohort
│  └─ Quick links: Content, AI, Operations
│
├─ Weekly Workflow:
│  ├─ Monday: /admin/scraper → trigger pipeline
│  │   └─ PIB + ARC + Lok Sabha fresh content
│  ├─ Tuesday: /admin/content → review AI-generated topics
│  │   └─ Approve / edit / reject with one-click actions
│  ├─ Wednesday: /admin/nudges → schedule weekly nudges
│  │   └─ Create, target, preview nudges
│  ├─ Thursday: /admin/hermes → review agent conversations
│  │   └─ Filter by user, flag issues
│  └─ Friday: /admin/analytics → export reports
│
└─ Alerts: Scraper failure, AI provider outage, payment anomaly
```

**Emotional Arc**: Responsible → In control → Efficient
